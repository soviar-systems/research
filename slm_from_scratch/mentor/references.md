# Course Reference Library

## 🛠 Classification Legend
This library uses the following tags to categorize resources. Use these to ensure the mentor provides the correct "bridge" based on the type of conceptual friction encountered.

- **[Primary]**: Authoritative gold standard/official definition (e.g., original papers, core textbooks).
- **[Intuition]**: Conceptual or visual explanations to help a concept "click" (e.g., videos, visualizations).
- **[Alternative]**: Rigorous but provides a different lens or explanation (e.g., a different professor's notes).
- **[Deep Dive]**: Advanced "Architect's Edge" (e.g., hardware optimizations, edge cases, NVIDIA blogs).
- **[Note]**: Meta-information, specific page pointers, or context on why a resource was added.
